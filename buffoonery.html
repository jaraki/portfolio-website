Buffoonery

4-person Cooperative Tetris

In order to appease the Mad King, four court jesters by the names of Sir Tet, Madame Pac, The Illusive Galag, and Kevin must work together to ‘Make Me Laugh’. The jesters MOVE, ROTATE, and STACK to fit into different positions. This may please the King, but only for a moment. So begins the buffoonery...

The Game

Buffoonery is a 4-player cooperative Tetris game made with Unity C# at the Global Game Jam 2024. This was one of the most polished games that I have been a part of for Global Game Jam.

The scoping of this project was extremely well-managed. We were able to create a simple game design that we were able to get working relatively early in the jam. This let us pursue a level of polish that usually does not happen during game jams.

Our Global Game Jam submission page can be found here.
